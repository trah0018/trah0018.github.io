Html Tags

html:
Defines the root element of an HTML doccument.
head:
It is the parent element.
body:
Contains the content of the HTML document, such as text, images, links, etc.
title:
Sets the title of the HTML document, which appears in the browser's title bar or tab.
li:
Defines a list item, used within ul or ol (ordered list) elements to represent items in the list.
